HTML
Definition: The core language of web pages,responsible for structuring and organizing content.
Role: Defines the content of a web page,such as headings,paragraphs,lists,images,links,and more.
Structure: Uses elements (enclosed in tags like (h1,p,img) and attributes (like src for image sources) to create a hierarchical structure.
Analogy: Think of HTML as the skeleton of a website,providing the basic framework.

CSS
Definition: A language used to style and design the appearance of web pages.
Role: Controls the visual presentation of HTML elements,including colors,fonts,layouts,spacing,and animations.
Structure: Uses selectors to target specific HTML elements and apply rules (like color: red;,font-size: 16px;) to modify their appearance.
Analogy: Think of CSS as the clothing and makeup that make the skeleton look presentable.

JAVASCRIPT
Definition: A programming language that adds interactivity and dynamic behavior to web pages.
Role: Makes web pages more engaging and responsive,allowing for user interactions,animations,form validation,data manipulation,and more.
Structure: Uses scripts to execute code within the browser,interacting with HTML elements and modifying their content or behavior.
Analogy: Think of JavaScript as the muscles and nervous system that bring the skeleton to life.
